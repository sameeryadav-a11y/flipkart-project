# 🛒 E-commerce Backend (Scalable Architecture)

This is a backend starter for a Flipkart/Amazon-like e-commerce application built with:

- Next step ready frontend (Next.js)
- Node.js + Express (TypeScript)
- Prisma ORM
- PostgreSQL
- Modular Monolith Architecture (Microservices-ready)

---

## 🚀 Project Goal

To build a scalable backend system that can **easily transition into microservices** in the future.
